# Deep Research Prompt Frameworks

Use one of these structures per prompt.

## 1) Strategy Framework
- Objective
- Context
- Constraints
- Evidence to gather
- Decision output format
- 30/60/90 action plan

## 2) Conversion Framework
- Funnel stage
- Audience segment
- Problem hypothesis
- Benchmark request
- Test design (A/B)
- KPI + expected lift

## 3) Offer/Positioning Framework
- ICP profile
- Alternatives/substitutes
- Message map
- Objection map
- Proof assets needed
- Recommended copy angles

## 4) Ops/System Framework
- Current workflow
- Failure points
- Automation options
- Cost/time impact
- Implementation checklist
- Monitoring setup

## 5) Content/SEO Framework
- Search intent clusters
- Content gaps
- Competitive SERP analysis
- Internal linking map
- Distribution plan
- Monetization tie-in
