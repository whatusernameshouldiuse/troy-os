# Memory Maintenance Runbook (Weekly)

## Promote
Move durable items to `MEMORY.md`:
- Decisions with long-term impact
- Stable preferences
- Reusable workflows
- Lessons that prevent repeated mistakes

## Keep in daily files only
- One-off chatter
- Temporary status updates
- Low-signal noise

## Weekly steps
1. Review last 7 daily memory files
2. Promote high-signal items to `MEMORY.md`
3. Remove stale/outdated items from `MEMORY.md`
4. Log updates in `mission-control/ops-log.md`
