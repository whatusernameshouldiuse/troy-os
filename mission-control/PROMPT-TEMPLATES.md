# Prompt Templates (Reusable)

Use these as defaults for repeatable high-quality execution.

## 1) Promo Build
**Objective:** Build a launch-ready promo package.
**Context:** [product/event/audience]
**Constraints:** [deadline/platform/brand voice]
**Plan:** concept -> assets -> copy -> schedule -> QA
**Output:**
- Campaign matrix
- Platform-specific copy
- Image/video prompt pack
- Production checklist

## 2) SEO Sprint
**Objective:** Improve organic performance for [page/topic].
**Context:** [site/current ranking/target keywords]
**Constraints:** [no dev changes / limited content / etc.]
**Plan:** audit -> prioritize -> implement -> measure
**Output:**
- Top opportunities list (impact/effort)
- On-page updates
- Internal link suggestions
- Tracking checklist

## 4) Debugging / Incident
**Objective:** Fix [error/issue].
**Context:** [system/logs/recent change]
**Constraints:** [risk tolerance/no downtime]
**Plan:** reproduce -> isolate -> fix -> verify -> prevent
**Output:**
- Root cause
- Patch
- Validation steps
- Postmortem note

## 5) Ops Audit
**Objective:** Audit workflow reliability.
**Context:** [agent/tooling/workflow]
**Constraints:** [time window]
**Plan:** health checks -> failure points -> mitigations
**Output:**
- Risk list
- Priority fixes

- Follow-up checklist
