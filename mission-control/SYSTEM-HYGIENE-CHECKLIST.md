# System Hygiene Checklist (Weekly)

Cadence: 1x/week (recommended Friday afternoon)

## 1) Workspace hygiene
- [ ] Duplicate file sweep (recent clutter)
- [ ] Archive low-signal temp outputs
- [ ] Confirm indexes/maps still accurate

## 2) Vault performance
- [ ] Obsidian responsiveness check
- [ ] Validate `.obsidianignore` still covering heavy dirs
- [ ] Identify new heavy folders (>200MB)

## 3) Agent ops health
- [ ] Run AGENT-OPS-PREFLIGHT
- [ ] Test sub-agent spawn + fallback path
- [ ] Confirm messaging channel health

## 4) Memory hygiene
- [ ] Promote durable learnings to MEMORY.md
- [ ] Keep daily memory concise and useful
- [ ] Prune stale/outdated guidance

## 5) Workflow compounding
- [ ] Identify repeated tasks (2+ in week)
- [ ] Create/update SOP drafts
- [ ] Propose new skill candidates

## Weekly output
Log results to `mission-control/ops-log.md` with:
- Date
- Issues found
- Fixes applied
- Open actions + owners
