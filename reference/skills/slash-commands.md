# Available Slash Commands

Quick reference for all installed skills. Type `/command` to invoke.

---

## Ralph (Autonomous Coding)

| Command | Purpose |
|---------|---------|
| `/ralph-clarify` | Ask 40-70 clarifying questions, save to `.claude/clarify-session.md` |
| `/ralph-plan` | Create PROMPT.md and TODO.md from clarify session |
| `/ralph-loop` | Execute tasks from TODO.md one at a time |

## Founder Advisors (from founder-gpts by Ryan Carson)

| Command | Role |
|---------|------|
| `/cfo` | Financial strategy, budgets, cash flow, break-even analysis |
| `/cto` | Technology infrastructure, architecture, security, scalability |
| `/cpo` | Product strategy, roadmap, market positioning, feedback loops |
| `/cro` | Revenue growth, sales strategy, conversion optimization, LTV:CAC |
| `/coach` | Founder mentoring, vision clarity, goal setting, challenges |
| `/investor` | VC perspective, due diligence, fundraising readiness, exit strategy |

**Usage:** Founder advisors push back when you don't know critical numbers. Use for strategic thinking or stress-testing business decisions.

## Tracking & Metrics

| Command | Purpose |
|---------|---------|
| `/session-end` | Log session to Shared Brain (cross-instance memory) |
| `/track` | Log session accomplishments to weekly file (public changelog) |
| `/weekly-summary` | Generate Twitter-ready weekly metrics & highlights |

**Usage:** Run `/track` at end of productive sessions to build your public changelog.
