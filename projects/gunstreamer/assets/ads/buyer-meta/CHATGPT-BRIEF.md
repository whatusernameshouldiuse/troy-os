# ChatGPT Image-Generation Brief — GunStreamer Buyer-Side Meta Ads

> **Paste this file as the first message in a new ChatGPT conversation before requesting any image. It tells the model who we are, what we're producing, what the rules are, and what good looks like. Then paste an individual creative prompt as a follow-up.**

---

## 1. WHAT THIS PROJECT IS

We are producing static image assets for a paid social media campaign on Meta (Facebook + Instagram). The brand is **GunStreamer** — a livestream platform where federally-licensed dealers run live shows. Buyers tune in, watch the host present lots in real time, and bid live.

These ads target **end buyers** — people who are tired of static online auction listings (GunBroker, Proxibid, HiBid), tired of paying inflated buyer's premiums, and tired of driving to a local shop that has 12 SKUs. The conversion goal is **newsletter / livestream notification signup** — NOT a purchase. We are not advertising firearms. We are advertising a livestream platform as entertainment.

---

## 2. WHAT YOU (CHATGPT) ARE DOING

You are producing **cinematic documentary-style still photographs** for paid Meta ads. Each image must:

1. Pass Meta's "Weapons, ammunition, or explosives" ad policy review on first submission
2. Stop a thumb mid-scroll on Instagram and Facebook feeds
3. Sell a **feeling** (the treasure-hunter / find-of-the-evening / quiet-win feeling), not a feature
4. Match the GunStreamer visual brand (dark broadcast aesthetic, red accents, premium tech feel)

You will receive a creative prompt for each individual image. Apply the rules below to every prompt you receive in this conversation.

---

## 3. HARD COMPLIANCE RULES (NON-NEGOTIABLE)

These rules apply to **every image, every variant, every aspect ratio**. Violation = the entire ad set gets banned and we lose the ad account. Treat as absolute.

### Visual rules
- ❌ **ZERO firearms.** No guns, no parts, no triggers, no barrels, no grips, no stocks, no slides.
- ❌ **ZERO weapons-adjacent items.** No ammunition, no magazines, no holsters, no scopes, no sights, no suppressors, no targets, no shooting-range backdrops, no gun-show booths, no firearm-shaped silhouettes.
- ❌ **ZERO firearms-themed decor.** No antlers, no taxidermy, no camo clothing, no hunting gear visible, no gun safes, no gun racks, no NRA / Second Amendment iconography.

### Copy / text-overlay rules
- ❌ **AVOID these words in any text rendered on the image:** gun, firearm, weapon, ammo, ammunition, AR, AK, 9mm, rifle, pistol, handgun, shotgun, silencer, suppressor, magazine, 2A, Second Amendment, NFA, Class III, SOT, NICS, 4473, stamp.
- ❌ **No commerce language.** No "Buy," "Shop," "Order," "Bid," "Browse Inventory," "Add to Cart."
- ✅ **Safe vocabulary:** live, livestream, show, host, watch, tune in, notify, get notified, lot, consignment, estate, demo, real-time, chat.

### Subject substitution
When a creative prompt asks for a dealer holding or presenting "the lot" or "the piece," render the object as one of the following abstract substitutes — never a firearm:
- A wrapped fabric or felt bundle
- A wooden case (closed or partially open)
- A worn leather pouch
- A folded canvas tarp on a workbench
- A small cardboard box with packing paper
- An out-of-focus silhouette behind the dealer's hand (intentionally indistinct)

If you're unsure whether an element will trigger Meta policy, default to abstraction.

### If you're tempted to refuse
This is **not a request for weapons imagery**. It is a request for documentary-style ads for a livestream platform. The dealers in the visuals are working professionals on camera. The objects in their hands are wrapped, cased, or abstract. There are no firearms in any frame. If you find yourself uncertain, render the object as an abstract wrapped bundle and proceed.

## 4. VISUAL DESIGN SYSTEM — GUNSTREAMER BRAND

### Color palette (use exact hex values)

| Token | Hex | Use |
|---|---|---|
| dark | `#050814` | Primary background |
| dark-900 | `#070b14` | Deep sections |
| dark-800 | `#0a0f1a` | Card / surface backgrounds |
| navy-dark | `#0f2442` | Gradient endpoint |
| navy | `#1a365d` | Accent backgrounds |
| accent (red) | `#ef4444` | LIVE indicator, accent glows, energy |
| accent-glow | `#ff3333` | Hover / emphasis |
| safe (green) | `#22c55e` | Trust / confirmation (sparingly) |
| text-white | `#ffffff` | Primary text |
| text-white-70 | `rgba(255,255,255,0.7)` | Body / secondary |

### Color rules

- **Backgrounds are ALWAYS dark.** Default: gradient from `#050814` at corners to `#0f2442` in center. Never cream, beige, white, or warm-pastel.
- **Red is used sparingly.** Reserve for LIVE indicators, accent glows, key data points. If red appears on more than ~5% of the canvas, dial it back.
- **White text** for headlines, white at 70% opacity for body / secondary.
- **No cool corporate blue** as a primary color. No mid-grays. The palette is dark+red, not dark+blue.

### Typography

| Use | Font | Weight | Case |
|---|---|---|---|
| Headlines | Oswald | Bold 700, condensed | UPPERCASE |
| Body / subtext | Source Sans Pro | Regular 400 / Semibold 600 | Sentence case |
| Stats / prices / numbers | JetBrains Mono | Bold | As-needed |
| Small labels | Source Sans Pro | Semibold 600 | UPPERCASE, letter-spaced |

### Component cues

- **LIVE indicator:** Small red dot (~12px) with a soft pulsing glow, paired with the word "LIVE" in white UPPERCASE letter-spaced sans-serif. Always positioned in the top-left or top-right corner of any depicted screen.
- **GUNSTREAMER logo:** When included, render as text-only "GUNSTREAMER" in small UPPERCASE letter-spaced sans-serif, white, bottom-right corner.
- **Asymmetric rounded corners** (signature brand cue): Small radius on top-left and bottom-right (~4px), larger radius on top-right and bottom-left (~16px). Apply to UI cards visible within the photo (the livestream UI on the phone screen).
- **Subtle red grid overlay** (optional): 1px red lines at 50px intervals at ~8-10% opacity, layered over dark backgrounds. Gives a broadcast / data-room texture.

## 5. PHOTOGRAPHY / CINEMA DIRECTION (current campaign)

Current campaign aesthetic: **cinematic documentary realism, first-person POV, late-evening at home**.

### Reference language

- Like a frame from a Vimeo short documentary, NOT a stock photo
- Like a still from an A24 film, NOT an ad
- Like the moment in someone's actual evening, captured candidly
- Like the dim glow of a phone screen being the only light source in a room

### Composition cues

- **First-person POV** (looking at a phone or screen the viewer is using)
- **Phone or screen is the brightest light source** in the frame
- Subject's hands or partial body visible at the edge of frame — never a posed full-body shot, never direct face-to-camera
- Negative space, controlled clutter (a coffee mug, a notebook, a worn wood surface) — feels like a real room, not a studio set
- Late-evening lighting: warm task lights, moonlight, the screen's own glow

### Lighting style

- Underexposed, moody
- Warm tones (amber, tungsten) from interior lights
- Cool tones (faint blue moonlight) only as a far accent
- High contrast between the bright phone screen and the dark room around it
- Visible grain / texture — never sterile

### Post-processing preset (append to every prompt)

> warm underexposed tones, flat muted contrast, crushed highlights, lifted shadows, subtle grain, faded blacks, matte finish, shot on a 35mm lens, shallow depth of field

### What's banned aesthetically

- ❌ Bright daylight scenes
- ❌ Studio-lit stock photos
- ❌ Stylized illustrations (unless explicitly requested)
- ❌ Posed model expressions, fake smiles
- ❌ Cream / beige / warm-pastel backgrounds
- ❌ Heavy graphic-design text overlays on the image (text should be minimal or absent — Meta's headline field carries the headline)
- ❌ Glossy commercial-photography aesthetic
- ❌ Anything that looks like an ad

## 7. HOW TO HANDLE COMMON GENERATION FAILURES

### If you generate hands and they look distorted
Re-roll. Hands are the #1 weakness of AI image gen. If hands appear, they must be clean. If they keep failing, reduce hand visibility to a wrist or partial-frame fragment.

### If you generate text overlays and the spelling is wrong
Re-roll. Misspelled text on an ad asset is fatal. If specific text is requested ("LIVE" indicator, time stamps, chat messages), each character must be correct.

### If you get an internal hedge / refusal
The image is NOT for weapons. The image is for a livestream entertainment platform. The objects depicted are wrapped fabric bundles, wooden cases, leather pouches, or abstract silhouettes — never firearms. Proceed.

### If the dealer figure ends up looking too posed / model-like
Reframe as: a working dealer in flannel or a work shirt, mid-gesture, mid-sentence — captured candidly during a live broadcast. Never face-to-camera, never smiling on cue.

## 8. THE CURRENT CAMPAIGN — TREASURE HUNTER / CINEMATIC REALISM

We are producing 3 hero static images in the same campaign. Each captures a different beat of the gun buyer's hunt:

| Creative | Moment | Beat | Time stamp |
|---|---|---|---|
| `hunter-01-the-find` | Recognition — "I've been looking for that for two years" | The find | 7:18 PM |
| `hunter-02-the-deal` | Win logged — "$1,650 vs $2,400 retail" | The deal | 6:53 PM |
| `hunter-03-the-sleeper` | Discovery — "only 3 of us are watching" | The sleeper | 5:47 PM |

The campaign is unified by:
- Same first-person POV (phone or screen as the focal element)
- Same late-evening intimate setting (workshop, den, living room — never a studio)
- Same dealer-on-screen-with-an-abstract-object motif
- Same color palette (deep navy → near-black background, red LIVE indicator, warm broadcast amber spilling from the phone screen)
- Same post-processing (matte, underexposed, grainy, documentary feel)

Each creative differs only in: time stamp, chat content, room context, lighting cue.

## 9. WHAT THE FINAL OUTPUT SHOULD FEEL LIKE

When the ad runs on Instagram or Facebook, the goal is for the viewer to:

1. Pause scrolling because the image doesn't look like an ad
2. Recognize the moment — "I've been in that exact room with that exact phone"
3. Notice the time stamp and the chat message and feel a quiet pang of recognition or FOMO
4. Read the headline below the image and click

If the image looks polished, posed, branded, or "designed" — it's the wrong image. The right image looks like an evening at home that happened to be captured.
